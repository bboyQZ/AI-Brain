# Chronicle · 公司录用与亲密互动规范

> 供所有女员工人物故事复用的世界观硬规则。尺度：**露骨亲密可写，色情不写（主 A + 少量 B）。**

## 面试：女性应聘者统一流程

乔志旗下公司——含恋爱体验馆、十秘、管理部、文艺部及全楼各岗位——对**所有女性应聘者**执行同一套**亲密互动测试**，不因部门、岗位或与张迪是否同学而例外。

测试在 HR 与秘书监督下进行，应聘者签署《面试须知》后自愿进入。可随时喊停；喊停即视为放弃录用。

| 环节 | 内容 |
|------|------|
| 牵手 | 测试掌心反应、是否躲闪 |
| 拥抱 | 测试身体贴近时的临场反应 |
| 接吻 | 含唇吻，评估配合度 |
| 舌吻 | 深吻计时，记录是否迎吻、是否腿软/失控 |
| 调戏 | 须能接受乔志面试中的言语调戏与肢体挑逗（捏下巴、耳边低语、腰际轻扣、隔衣抚摸等） |
| **可选 · 面试间发生关系** | 视人物剧情与临场配合度，面试间**可以**直接发生性关系（走「主 A + 少量 B」尺度，不写过程） |

恋爱体验馆一线接待岗额外强调「吻技」——因岗位卖的是真实亲密反馈，纸上简历无法替代身体临场。

## 录用终局：全员女朋友

**所有女员工，最终皆成为乔志的女朋友。**

- 关系可**面试当场**确认（如张迪：舌吻后录用、当晚确立）
- 也可**入职后**在同事见证或私密场景中落定（如秘书、保洁、网红岗——契机各异）
- 终点一致：**员工身份 + 女朋友身份**并存
- **李云浩为女性员工**，同样在「全员女友」终局之内；员工中不设男性例外

乔志从不「只招人不动心」——公司缺人，更缺能与他发生真实情感牵连的人；亲密测试既是筛选，也是关系入口。

## 确立关系后：叙事尺度（主 A + 少量 B）

**适用阶段**：**任何**发生性行为的场景，无论面试间、办公室、宿舍、大学时期与乔志的私密接触——一律走本档，不写过程。

所有女员工人物故事的亲密描写，默认遵循本档。

### A 档 · 主写法（结果一句带过）

交代**时间 + 结果**，不描写性行为过程：

- ✅ 「那晚他们上了床。」
- ✅ 「入职一周后，她第一次把身体交给他。」
- ✅ 「那是她的第一次；过程她不愿细写，只记得终于不用偷。」
- ❌ 性器官、插入、体位、节奏等任何过程细写

### B 档 · 少量点缀（前戏氛围，宜短）

可写**脱衣、挑逗、气氛**，停在「将要」或接到 A 档一句：

- ✅ 解扣子、褪外套、吻到床边、耳边低语
- ✅ 手在腰/背/腿外侧、呼吸交缠、腿软、主动贴近
- ✅ 乔志的调戏台词（与面试调戏同风格，更私密）
- ❌ 写成以性刺激为主的长段；❌ 接到 C 档过程描写

### 推荐段落结构

1. 情感节点（坦白、吃醋、确认关系等）  
2. **少量 B**：脱衣 / 挑逗，2～5 句  
3. **主 A**：「那晚他们发生了性关系」类一句，或无过程的结果句  
4. 切回心理 / 关系（如「不再是小偷」「全员女友里的她」）

### C 档 · 禁止

- 性器官直白描写、性行为过程细写、以读者性兴奋为目的的长段黄文

## 写作引用提示

写任意女员工故事时可套用：

1. 她如何完成（或差点放弃）亲密互动测试  
2. 她与乔志确立女友关系的**具体时间/场景**（可与张迪不同）  
3. 她的「特殊之处」在于暗恋多久、执念形式、部门职能——**而非**「是不是女友」（女友是终局，不是彩蛋）  
4. 若写**第一次上床**等：用 **主 A + 少量 B**（见上一节），勿与面试舌吻混在同一场景

## 尺度红线（速查）

| 场景 | 可写 | 不可写 |
|------|------|--------|
| 面试间 | 牵手～舌吻、调戏；可发生性关系（主 A + 少量 B） | 性行为过程细写、性器官直白 |
| 确立关系后 | 主 A 结果句 + 少量 B 脱衣挑逗 | C 档过程细写 |
| 大学暗恋线 | 偷闻/偷物、生理暗示、夹腿；**允许**与乔志发生实际性行为（视人物剧情，仍走主 A + 少量 B） | C 档过程细写 |
