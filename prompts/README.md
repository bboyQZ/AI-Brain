# `prompts/` — 模型提示词

大模型「人设 + 答题规则」放这里，**不要**再写回 `rag_service.py` 长字符串。

## 文件

| 文件 | 用途 |
|------|------|
| `rag_system.txt` | Chat API 的 `role: system` |
| `rag_user.txt` | 用户侧模板：塞入检索资料 + 当前问题 |

## `rag_user.txt` 里的占位符

必须保留（代码里 `.format` 填充）：

- `{context}` — 检索到的知识片段  
- `{query}` — 用户本轮问题  

删掉占位符会导致报错或资料接不进去。

## 怎么生效

`app/services/rag_service.py` 按文件修改时间缓存；**保存后下一次对话**就会用新内容，一般不用重启后端。

若你刚改完完全没变化：确认保存的是这两个文件，并再发一条新消息。

## 当前策略（简要）

- 优先根据知识库回答并引用  
- 资料不足时允许通用知识补充  
- 补充须标注：`【补充说明】（非知识库原文）`  

改风格、松紧、引用格式：优先只改本目录文本。
